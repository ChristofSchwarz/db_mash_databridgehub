This mashup brings important new features around (re-)deployments of apps and more. The key part is the possibility to refresh apps from a given source app in parts: any combination of design, data and script can be copied, whereas the QMC (Qlik Management Console) only allows to replace the app in one (all parts together).
More help see readme.md in this Github Repository.

You can call this hub.html with url querystrings:
stream= guid of the stream to open, or "mywork" to show unpublished apps
app= guid of the app for which to open the dialog. The stream, in which the app is in (or "My Work" if unpublished), will also be selected
from= guid of the app from which design, script, and/or data will be copied from
importdesign pre-selects the "Import design" checkbox, no value needed, just add importdesign to the url
importdata pre-selects the "Import data" checkbox, no value needed
importscript pre-selects the "Import script" checkbox, no value needed
